# P2P Arbitrage Scanner Bot для Bybit

Бот для пошуку арбітражних можливостей на P2P-ринку Bybit (USDT/UAH, Monobank).

## Функціонал

- 🚀 Сканування P2P ринку кожні 5 секунд
- 💰 Розрахунок чистого прибутку та ROI
- 📱 Telegram сповіщення про знайдені можливості
- 📊 Веб-дашборд з аналітикою
- 💾 Збереження історії у SQLite/PostgreSQL
- 🛡 Захист від rate limits
